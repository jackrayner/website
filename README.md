This site is currently using the Jekyll [minima theme](https://github.com/jekyll/minima).

### Pre-requisites
```
$ brew install ruby
$ export PATH="/usr/local/opt/ruby/bin:$PATH"
```

### Running the Jekyll server

```
$ cd website/
$ make serve
```
